# Roadmap

## Milestone 0: Repository Baseline

- Preserve the current Riddle-based device prototype.
- Keep device scripts, launcher notes, and Mac helper tools in one repo.
- Exclude secrets, SDKs, build outputs, and unconfirmed font binaries.

## Milestone 1: Configurable Server

- Add a tiny server with `/turn`.
- Move model provider configuration out of the tablet.
- Add prompt profiles for language, age level, answer length, and teaching tone.
- Return structured JSON for phone and tablet outputs.

## Milestone 2: Phone Companion

- Build a mobile web page for push-to-talk.
- Add speech-to-text input.
- Add text-to-speech playback.
- Show the same turn history the tablet is rendering.

## Milestone 3: Knowledge and Memory

- Add a small knowledge base for family materials and learning content.
- Store long-term child memories server-side.
- Make answers cite or follow retrieved knowledge instead of drifting.

## Milestone 4: Tablet Integration

- Change the Riddle prototype from direct model calls to DouDou server calls.
- Render structured paper cards.
- Support simple drawing prompts and line-art responses.
